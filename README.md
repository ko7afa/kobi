# 🃏 كوبي — اللعبة الليبية الأونلاين

لعبة كوبي الليبية متعددة اللاعبين — 4 لاعبين في الوقت الفعلي عبر WebSocket.

---

## 🚀 كيف تشغل اللعبة محلياً

```bash
# 1. ثبت المكتبات
npm install

# 2. شغل السيرفر
npm start

# 3. افتح المتصفح
http://localhost:3000
```

---

## ☁️ رفع على سيرفر (Render.com — مجاني)

1. ارفع المشروع على GitHub
2. اذهب إلى [render.com](https://render.com) وأنشئ حساب
3. اضغط **New Web Service**
4. اربطه بـ GitHub repository
5. الإعدادات:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Port:** `3000`
6. اضغط **Deploy** — الموقع يشتغل خلال دقائق!

---

## ☁️ رفع على Railway

```bash
npm install -g railway
railway login
railway init
railway up
```

---

## 🎮 كيف تلعب

1. افتح الموقع وأدخل اسمك
2. اضغط **"أنشئ غرفة جديدة"**
3. شارك الرابط عبر **واتساب** مع 3 أصدقاء
4. اللعبة تبدأ تلقائياً بمجرد انضمام 4 لاعبين

---

## 📋 قوانين كوبي الليبي

- **4 لاعبين** — كل واحد يأخذ 13 ورقة
- **القلب (كوبي) = طرنيب دائم** يكسب على أي بذلة أخرى
- **اللفة الأولى:** من يجمع 3+ ديار يسجل، أقل من 3 = سالب 3
- **اللفات التالية:** نظام الطلبات حسب نقاطك
- **خنقة المجيرة:** Q قلب + A/K قلب = +5 للقاتل، -5 للضحية
- **الفوز:** 21 نقطة أو أكثر
- **الخسارة:** -11 نقطة أو أقل

- **Real-time:** WebSocket للعب الفوري
